test.anti_deplacement_contraire(mode, x, y, liste, antecedent, fct)[source]¶

fonction qui empeche un mouvement contraire au deplacement précedent. En effet grace à cette fonction si joueur 1 joue un coups en haut et deplace les pions de joueur 2,joueur 2 n’a pas le droit d’effectuer aussitot un mouvement contraire à celui de joueur 1 (ici un coup en bas dans les même case verticales). cette fonction possedent deux “modes”, w pour write ou on inscrits les mouvement ainsi que les position effectuer par le joueur, t pour test pour verifier si le joueur peut jouer le coup qu’il demande.

Parameters:
  • mode (chaine de caractére.) – permet deux differencier le mode t du mode w.
  • x (entier entre 0 et 6.) – donne la position x du pions selon la grille.
  • y (entier entre 0 et 6.) – donne la position y du pions selon la grille.
  • liste (liste.) – contient la les element constituer selon le mouvement réalise (si l’on travaile sur les x ou les y) des element y des sous-liste de plateau ou bien d’une sous-liste de plateau.
  • antecedent (dictionnaire.) – contient la position, le mouvement ainsi que la liste du joueur qui à jouer juste avant.
  • fct (chaine de caractére) – parametre d’identifier la direction dans laquele on veut deplacer ses boules
Returns:

Un booléen permetant de dire si oui ou non le mouvement est autorisée.
Return type:

Booléen.
